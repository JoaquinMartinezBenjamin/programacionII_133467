101.- Se tiene un conjunto de 1,000 tarjetas cada una contiene la
 información del censo para una persona:
1.- Número de censo,
2.- Sexo
3.- Edad
4.- Estado civil (a.- soltero, b. Casado, c. Viudo, d. Divorciado )
Diseñe un pseudocódigo estructurado que lea todos estos datos,
 e imprima el número de censo de todas las jóvenes solteras 
que estén entre 16 y 21 años.
